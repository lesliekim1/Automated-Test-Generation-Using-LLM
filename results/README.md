# Results Directory

This directory contains all outputs from the experiment, including CSV results and screenshots.

## Structure

### Folders

* `prompts_results_csv/`
  CSV files containing complete results for each prompt.

* `statistical_tests/`
  Screenshots of statistical tests outputted by p10_statistical_analysis.py.

* `tables/`
  Screenshots of the tables outputted by p08_analysis.py. 

### Other

* `DEEPSEEK_combined_passed_results.csv`
  Combined results for the DeepSeek-Coder model, including only tests that passed all filters.

* `LLAMA_combined_passed_results.csv`
  Combined results for the Llama model, including only tests that passed all filters.

* `results.csv`
  A dataset containing baseline coverages only (before test generation).
